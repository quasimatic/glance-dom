# Installation

### Browser Example
```javascript
<script src="http://quasimatic.org/glance-dom/dist/glance-dom.js"></script>
```

#### Usage

```javascript
var element = glanceDOM("click me");
```

### Node Example

```shell
npm install glance-dom
```

#### Usage

```javascript
var glanceDOM = require("glance-dom").default;
var element = glanceDOM("click me");
```
